# Vanilla JS Task Manager

My first project built using HTML, CSS, and JavaScript.

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Move completed tasks to a completed section
- Interactive hover actions

## Concepts Practiced

- DOM Manipulation
- Event Handling
- Dynamic Element Creation
- Class Manipulation
- CSS Grid
- Responsive Layout Techniques

## What I Learned

- Working with dynamically created DOM elements
- Attaching event listeners to generated elements
- Managing UI state with JavaScript
- Structuring a small frontend application

## Future Improvements

- Local Storage support
- Better edit workflow
- Task categories
- Priority levels
- Responsive mobile layout

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
